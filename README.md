## Getting Started

### Install
Install via npm:
```console
npm install @this-empathy/locales-patterns
```

Install via yarn:
```console
yarn add  @this-empathy/locales-patterns
```

### How to use
#### Import

```javascript
import Locales from ' @this-empathy/locales-patterns'
```
